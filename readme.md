# Advanced Content Manager

A modular WordPress plugin for managing and displaying filterable content using
custom post types, meta fields, AJAX filtering, and Gutenberg blocks.

## Features
- Custom post type (Resources)
- Custom taxonomies
- Secure custom meta fields
- AJAX-powered filtering
- Server-side rendered Gutenberg block
- REST API ready

## Requirements
- WordPress 6.0+
- PHP 8.0+

## Installation
1. Clone this repository into `wp-content/plugins`
2. Activate the plugin
3. Add the Resource Grid block or `[acm_resource_grid]` shortcode
